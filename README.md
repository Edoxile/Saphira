Saphira
-------

A perl-based IRC bot.

This page should contain information about how to work with it somewhere in the future...

Modules
=======

These are the modules needed by this bot to run:

### Common ###

- Config::IniFiles
- POE::Kernel
- POE::Component::IRC
- DBI
- DBD::mysql
- POE::Component::SSLify

### Title ###
- URI::Title

### Lastfm/wikipedia ###
- JSON::XS

### Google ###
- Google::Search
- HTML::Entities
- Encode

### WolframAlpha ###
- WWW::WolframAlpha

### Interactive ###
- Switch
